# Web-Final

Full-stack web application — Course Final Project.

## Tech Stack

- **Backend**: Node.js + Express + TypeScript + MongoDB
- **Frontend**: React + Vite + TypeScript

## Project Structure

```
Web-Final/
├── backend/        ← Node.js + Express API
│   └── src/
├── frontend/       ← React + Vite client
│   └── src/
└── README.md
```

## Getting Started

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Copy `backend/.env.example` to `backend/.env` and fill in your values.
